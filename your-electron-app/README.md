# Electron 應用程式

這是一個使用 Electron 框架開發的桌面應用程式。

## 功能特色

- 基於 Electron 的跨平台桌面應用
- 包含加密功能模組
- 支援多個 HTML 頁面

## 安裝

```bash
npm install
```

## 執行

```bash
npm start
```

## 專案結構

- `main.js` - Electron 主程序
- `preload.js` - 預載腳本
- `renderer.js` - 渲染程序腳本
- `crypto.js` - 加密功能模組
- `index.html` - 主頁面
- 其他 HTML 檔案 - 應用程式的各個頁面

## 技術棧

- Electron
- Node.js
- HTML/CSS/JavaScript
